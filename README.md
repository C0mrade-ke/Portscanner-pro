# PortScanner Pro
 by c0mrade

> A simple Python-based TCP port scanner with a hacker-style terminal interface.

---

## Features

- Scans common network ports
- Detects open services
- Fast TCP connection check
- Interactive scanning loop

---

## Installation

Clone the repository:

git clone https://github.com/c0mrade-ke/portsc-pro.git

Move into the directory:

cd portsc-pro

Run the scanner:

python3 portsc.py
